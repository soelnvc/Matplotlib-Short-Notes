# 📊 Matplotlib Short Notes for Everyone

A beginner-friendly, structured guide to mastering data visualization in Python. This repository covers everything from basic line graphs to advanced subplots and Pandas integration.

## 🗺️ Roadmap
1. **[Getting Started](./Notes/start.ipynb)**: Introduction to the Pyplot API, Data Generation, and basic styling.
2. **[Types of Charts](./Notes/typesofchart.ipynb)**: Deep dive into Bar charts, Pie charts, Scatter plots, and Histograms.
3. **[Mastering Subplots](./Notes/subplots.ipynb)**: Learning the **Object-Oriented (OO) API** to create multi-plot dashboards.
4. **[Pandas Integration](./Notes/pandaswithplt.ipynb)**: Visualizing real-world datasets and mapping categorical colors.
